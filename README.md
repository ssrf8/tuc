# 自用图片帖子网站

一个无需登录的自用图片管理网站。可以创建帖子、给帖子上传图片、浏览图片、删除单张图片或删除整个帖子。

## 启动

```bash
docker compose up -d --build
```

打开：

```text
http://localhost:3000
```

## 数据保存位置

默认使用当前目录下的 `data/`：

- `data/posts.json`：帖子和图片元数据
- `data/uploads/`：上传的图片文件

只要保留 `data/` 目录，重建容器不会丢数据。

## 配置

在 `docker-compose.yml` 中可以调整：

- `3000:3000`：左侧端口是宿主机访问端口
- `MAX_UPLOAD_MB=25`：单张图片最大上传大小，单位 MB

## 注意

这个项目按“自用、无需登录”设计。不要直接暴露到公网，除非你在前面加了访问控制，例如反向代理 Basic Auth、VPN 或内网访问限制。
